## Getting Started

### Dependencies

* .NET 5.0
* any platform where .NET 5 is a thing

### Usage

* add LibTrie to your project
* create a new ```Trie``` instance with ```var trie = new Trie();```
* add new words to it with ```trie.AddWord("example");```
